## О платформе

**Hammerhall** — это не набор разрозненных упражнений, а семья независимых образовательных проектов — **кузниц**. Каждая кузница посвящена одной теме и ведёт от простых приёмов к проектированию настоящих систем.

Внутри каждой кузницы — последовательная траектория из семи уровней сложности, единая предметная область вместо учебных примеров и автоматическая проверка решений. Студент не заучивает методы стандартной библиотеки, а учится **мыслить инженерно**.

> 💡 Не изучайте API. Учитесь мыслить.

---

## Как устроена кузница

Каждая кузница строится по единому стандарту платформы:

- **семь уровней** — от Easy до Grandmaster, плавный градиент сложности;
- **единая предметная область** — задачи складываются в общую траекторию, а не в россыпь примеров;
- **контракт задачи** — условие, эталон, модели и тесты согласованы между собой;
- **автоматическая проверка** — открытые тесты для разработки, закрытые для подтверждения;
- **измеримая сложность** — у каждой задачи свой вес, уровни сбалансированы по данным, а не на глаз.

Студент, переходя между кузницами, не переучивается заново — структура везде одинакова.
